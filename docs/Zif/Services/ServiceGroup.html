Overview

Register a group of Zif::Services.

Game automatically saves an instance of this class at the $services global variable, and registers the typical services (InputService, ActionServiceâ€¦) on it for you. It's also accessible through the Game instance via Game#services.

Examples:

Invoke a method on the TickTraceService via the ServiceGroup instance on a Game

$game.services[:tracer].mark("Hello World")
