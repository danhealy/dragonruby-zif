Overview

A mixin for automatically definining #serialize based on instance vars with setter methods If you have circular references in ivars (Foo.bar <-> Bar.foo), make sure one of the classes overrides #exclude_from_serialize and specifies the attr.

def serialize
  attrs = {}
  instance_variables.each do |var|
    str = var.to_s.gsub('@', '')
    next if exclude_from_serialize.include? str

    attrs[str.to_sym] = instance_variable_get var if respond_to? "#{str}="
  end
  attrs
end
